The Current Draft of Igor Gorlov's Coding Style Documentation

Natural Language

The official language of our community is English. It should be used in almost every case where natural language is required, including the documentation, file and directory names, identifiers within the source code, commit messages, interaction between the community members, and so on. In personal communication with me, you can also use Russian.

Generic Instructions

The following documents are designed to be read sequentially, in the presented order: each of them may contain information critical for understanding the next one.

General Guidelines
Provides a set of rules applicable to all possible projects regardless of their type or programming languages their are written in.
Document Requirements
Establishes the design of technical documents like itself: visual, semantic, and grammatical aspects are covered.
Version Control Guide
Defines a standard approach to project history development and maintenance.
