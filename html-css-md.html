Igor Gorlov's Style Guide for HTML, CSS, and Markdown

Source code written in the languages listed within the title is controlled by Prettier. You only need to configure this tool according to the following table (a config file at the top level of a worktree is recommended as the simplest and the most flexible solution). Please do not adjust the formatting manually, even if an automatically processed document looks somewhat abnormal — let Prettier take full responsibility for HTML, CSS, and Markdown code.

The Required Configuration for Prettier
Option Value
bracketSameLine true
bracketSpacing false
htmlWhitespaceSensitivity css
printWidth 120
proseWrap always
tabWidth 4
trailingComma none
useTabs true
