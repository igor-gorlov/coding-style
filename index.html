Welcome to Igor Gorlov's Coding Style Documentation

In all times, people have been needing to unify their goals and methods in order to collaborate effectively. The legend of the Tower of Babel can be considered an example of the contrary approach that had led to an epic fail. The simplest (and probably the most reliable) way to accomplish unification is defining common standards everyone should follow. This collection of documents is nothing but a try to maintain consistency and readability of the code we write together. The rules present here are not dogmas, but I hope you will take them seriously anyway.

Natural Language

The official language of our community is English. It should be used in almost every case where natural language is required, including the documentation, file and directory names, identifiers within the source code, commit messages, interaction between the community members, and so on. In personal communication with me, you can also use Russian.
