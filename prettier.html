Igor Gorlov's Guide to Formatting with Prettier

Prettier is a code formatter for HTML, CSS, Markdown, JavaScript, and JSON. It enforces its own coding style, which is not so perfect but consistent. All you need is a .prettierrc file containing Configuration options. This file should be located at the top level of your project and must be controlled by your VCS.

Please do not adjust the formatting manually, even if an automatically processed file looks somewhat abnormal — let Prettier take full responsibility for code representation.
